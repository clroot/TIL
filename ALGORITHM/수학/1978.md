# BOJ 1978 소수 찾기

- [문제 링크](https://www.acmicpc.net/problem/1978)

## 문제 요약

- 1~1000 범위의 숫자 N개를 입력 받아서
- 입력 받은 수 중에 소수가 몇개 있는지 구하기

## 구현

```python
# 사전에 소수들 구한 코드
from math import sqrt
prime = []
for i in range(2, 1000 + 1):
    is_prime = True
    for j in range(2, int(sqrt(i)) + 1):
        if i % j == 0:
            is_prime = False
            break

    if is_prime:
        prime.append(i)

print(prime)

```
